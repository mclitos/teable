# teable

Comandos utilizados:
``` 
git clone https://github.com/mclitos/teable
``` 
``` 
cd teable
``` 
Si no es para pruebas, Cambiar en .env los nombres de la base de datos 
```
POSTGRES_DB=(Combiar)
POSTGRES_USER=(Combiar)
POSTGRES_PASSWORD=(Combiar)
```
```
docker compose up -d
```

Ver en: "http://IP:7880"
